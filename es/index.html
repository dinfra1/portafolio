Mis Tecnologias

HTML

Es un lenguaje que permite definir y darle estructura a las páginas web.

CSS

Es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un usuario las visita.

JavaScript

Es un lenguaje de programación interpretado y orientado a objetos, pensado para agregar potencial de interacción y dinamismo a las páginas web.

ReactJs

Librería para crear interfaces de usuarios con código abierto, es decir, permite reutilizar recursos e incluso hacer colaboraciones.

NodeJs

Ideado como un entorno de ejecución de JavaScript orientado a eventos asíncronos, Node.js está diseñado para crear aplicaciones network escalables

PostgresSQL

PostgreSQL es un sistema de gestión de base de datos relacional open source de muy alta disponibilidad, gratuito y libre.

GitHub

GitHub es una forja para alojar proyectos utilizando el sistema de control de versiones Git.

Python

Lenguaje orientado a objetos, fácil de interpretar y con una sintaxis que permite leerlo de manera semejante a como se lee el inglés

C++

C++ es un lenguaje de programación diseñado en 1979 por Bjarne Stroustrup.
